# Technology Stack

## Backend
- **Framework**: FastAPI (Python)
- **Runtime**: Python 3.12+
- **Database/Auth**: Firebase Admin SDK
- **Validation**: Pydantic
- **Key Libraries**:
  - `uvicorn` (server)
  - `python-jose` (JWT)
  - `passlib` (Hashing)
  - `httpx` (Async HTTP)

## Simulation
- **Framework**: React
- **Runtime**: Node.js
- **Orchestration**: `concurrently`
- **Structure**: npm workspaces

## Staff Desktop App (CrysisDesk)
- **Framework**: Tauri v2 + React 19
- **Build Tool**: Vite 7
- **State Management**: Zustand
- **Real-time**: Socket.io-client

## Responder Desktop App
- **Framework**: Tauri v2 + React 19
- **Build Tool**: Vite 8
- **Real-time**: Socket.io-client

## Warden Android App
- **Language**: Kotlin
- **UI Framework**: Jetpack Compose
- **Networking**: Retrofit, OkHttp
- **AI**: Google Generative AI (Gemini) SDK
- **Build System**: Gradle (Kotlin DSL)
