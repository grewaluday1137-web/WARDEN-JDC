# CrisisSync Web Dashboard

This workspace contains the frontend Mission Control interface for the CrisisSync Simulation platform. 
Built on Next.js 15+, React 19, and Tailwind CSS.

## Getting Started

1. Go to the project root and ensure all dependencies are installed:
   ```bash
   npm install
   ```

2. Start the integrated dev server from the root:
   ```bash
   npm run local
   ```
   Or specifically start just the frontend:
   ```bash
   npm run dev -w apps/web
   ```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the Mission Control dashboard.
